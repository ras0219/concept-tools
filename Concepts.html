Description

This module defines data types and algorithms for abstractly manipulating concepts. The exported functions are meant to be used as

satisfies :: Set Concept -> Set Concept -> Bool

Checks if the first concept set satisfies the requirements of the second. Will fully simplify both arguments. Ex. 

>>> let i = Type "I"
>>> satisfies (singleton $ ForwardIterator i) (singleton $ InputIterator i)
True
